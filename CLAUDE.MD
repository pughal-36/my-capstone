## Project
Decision Journal — log decisions with reasoning/confidence, revisit later 
to record actual outcomes.

## Stack
- React 18 + Vite
- Plain CSS (or Tailwind — decide once UI work starts)
- No backend yet — local state / localStorage for now

## Conventions
- Commit messages: Conventional Commits (feat:, fix:, docs:, chore:, refactor:, test:)
- Components: functional components with hooks, no class components
- File naming: PascalCase for components (e.g. DecisionForm.jsx)

## Commands
- `npm install` — install dependencies
- `npm run dev` — start local dev server
- `npm run build` — production build
